# Paymob

Paymob ruby sdk

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'paymob'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install paymob

## Usage
First argument is key
```ruby
require 'paymob'
`Paymob.calc_secure_hash 'test', {'a': '1', 'b': '2', 'c': ''}`
```
